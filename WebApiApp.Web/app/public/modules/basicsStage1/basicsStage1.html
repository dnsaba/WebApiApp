Variables and Datatypes

What are variables?

A variable is used to store a value.

For example:

int myVar = 1;

Here, the variable is called "myVar" and it has a value of 1.

What are datatypes?

Datatypes are the different kinds of values that a variable can hold.

In our first example, we declared and initialized the variable myVar by typing int myVar = 1;

Here, the datatype is an integer (int) which is a whole number.

Other data types are:

  1. char - single characters like c, $, *
  2. ex) char example1 = 'a';
  3. string - all string of characters of any length
  4. ex) string example2 = "This is a string!";
  5. bool - true or false
  6. ex) bool example3 = true;
